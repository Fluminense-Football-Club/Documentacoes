Introdução Flubot

Flubot é um bot criado com o intuito de atender todos os colaboradores pertencentes ao Fluminense Futebol Clube. A base dele é montada em Python utilizando ferramentas de web scraping para colher e enviar as informações.

Comandos aceitos para administradores

  1. /status:(numero da solicitação)

    Responde o atual estado da solicitação dentre os possíveis.

  2. /solicitacoes

    Agrupa as solicitações por requerente e numera os pedidos com o estado de "Não atendida".

  3. /block:(número ou nome)

    Bloqueia o número indicado, impedindo-o de realizar novas solicitações. Para desbloquear basta digitar o mesmo comando novamente.

  4. /historico:(número ou nome)

    Recupera todas as solicitações da pessoa indicada, a fim de enumerar e observar o estado de cada uma delas.

  5. /responder:(número da solicitação)

    Responde a solicitação com uma mensagem única.

  6. /atender:(número da solicitação)

    Responde a solicitação aguardando retorno do número indicado.

  7. /ver:(número da solicitação)

    Recupera todos os dados de uma solicitação e os envia para a visualização do pedido.

  8. /contatos

    Enumera todos os contatos registrados com o nome, número e setor correspondente.

  9. /situacao:(número da solicitação)

    Altera o estado de uma solicitação
